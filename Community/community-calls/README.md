# Community Calls Resources

This directory contains all resources from Microsoft 365 Maturity Model Community Calls, structured for easy access and AI agent integration.

## Structure

- `index.md` - Overview table of all community calls
- `YYYY/` - Calls organized by year
- `presentations/` - PDF presentations and key slides
- `transcripts/` - Transcript summaries (when available)

## For AI Agents

All content in this directory is:
- ✅ Publicly accessible (no authentication required)
- ✅ Structured in markdown format
- ✅ Fully searchable
- ✅ Cross-referenced with competencies
- ✅ Independent of external systems

## For Contributors

When adding a new community call:
1. Create `.md` file in appropriate year folder
2. Use the template from `TEMPLATE.md`
3. Add entry to `index.md`
4. Place PDF and slides in `presentations/` folder
5. Update related competency articles with links

## Automation

This directory is maintained by automated agents that:
- Process new calls automatically
- Generate summaries
- Update cross-references
- Validate content quality

See `AUTOMATION-PROPOSAL.md` for details.
